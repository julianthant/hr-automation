# JSON Schemas

Generated by `npm run schemas:export` from each workflow's Zod input schema.

Wire these into VSCode's `yaml.schemas` setting (in `.vscode/settings.json`) to get
autocomplete + validation while editing batch YAML files, e.g.:

```jsonc
{
  "yaml.schemas": {
    "./schemas/emergency-contact.schema.json": "src/workflows/emergency-contact/**/batch*.yaml"
  }
}
```

The `*.schema.json` files here are gitignored — re-run `npm run schemas:export`
after any schema change.
