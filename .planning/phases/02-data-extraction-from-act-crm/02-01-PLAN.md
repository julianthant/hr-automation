    1. Create `src/extraction/search.ts`:
       - Import Page from "playwright" and log from "../utils/log.js"
       - Import ExtractionError from "./types.js"

       - `searchByEmail(page: Page, email: string): Promise<void>`:
         - Navigate to ACT CRM onboarding portal: `await page.goto("https://act-crm.my.site.com/hr/a1Z/o", { waitUntil: "domcontentloaded", timeout: 15_000 })`
         - Find search input using fallback chain: `page.getByPlaceholder("Search").or(page.getByRole("searchbox")).or(page.locator('input[type="search"]'))`
         - Fill with email, press Enter
         - Wait for results: `page.waitForLoadState("networkidle", { timeout: 15_000 })`
         - Log step messages (field names only, never the email value -- PII safe)
         - All selectors marked with `// SELECTOR: may need adjustment after live testing`

       - `selectLatestResult(page: Page): Promise<void>`:
         - Locate result rows: `page.locator("table tbody tr").or(page.locator('[role="row"]'))`
         - If count is 0, throw ExtractionError("No search results found")
         - Iterate rows, extract date text from last cell of each row
         - Parse dates with `new Date(dateText.trim())`, track latest index
         - Click the row with latest date
         - Wait for domcontentloaded after click
         - Mark date column selector with `// SELECTOR: date column position may vary`

    2. Create `src/extraction/navigate.ts`:
       - Import Page from "playwright" and log from "../utils/log.js"

       - `navigateToEntrySheet(page: Page): Promise<void>`:
         - Look for "UCPath Entry Sheet" using fallback chain: `page.getByRole("link", { name: /UCPath Entry Sheet/i }).or(page.getByText("UCPath Entry Sheet")).or(page.getByRole("tab", { name: /UCPath Entry Sheet/i }))`
         - Click first match with timeout 10_000
         - Wait for networkidle after click (timeout: 15_000)
         - All selectors marked with SELECTOR comment

    3. Create `src/extraction/extract.ts`:
       - Import Page from "playwright" and log from "../utils/log.js"

       - Define FIELD_MAP as `Record<string, string[]>` mapping each EmployeeData field name to an array of possible label strings (from research: Position Number/Position #/Position No, First Name/Legal First Name, Last Name/Legal Last Name, SSN/Social Security/Social Security Number, Address/Street Address/Address Line 1, City, State, Postal Code/Zip Code/ZIP, Wage/Pay Rate/Salary/Compensation, Effective Date/Start Date/Hire Date)

       - `extractField(page: Page, label: string): Promise<string | null>` (not exported):
         - Strategy 1 (label-based): `page.locator(\`text="${label}"\`).locator("xpath=..").locator("span, dd, td").last()`
         - Strategy 2 (ARIA): `page.getByLabel(label)`
         - Strategy 3 (table cell): `page.locator(\`td:has-text("${label}")\`).locator("xpath=following-sibling::td[1]")`
         - Try each with 3_000ms timeout, return first non-empty result or null
         - Mark all selectors with SELECTOR comment

       - `extractRawFields(page: Page): Promise<Record<string, string | null>>`:
         - For each entry in FIELD_MAP: log.step(`Extracting ${fieldName}...`) -- field name only, never the value (PII safe)
         - Try each label variant via extractField until one returns non-null
         - Return raw record

         f. On ExtractionError: log.error(error.message) -- do NOT log raw data (PII). Close browser, exit 1
